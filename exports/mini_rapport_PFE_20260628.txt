3. FACTEURS DE RISQUE ENVIRONNEMENTAUX  (Tableaux 3 et 4)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  3.1  Analyse univariée (OR bruts, Test exact de Fisher)
       20/24 variables significatives (p < 0,05)
       • Sucreries                           OR = 0.08 (0.06–0.11)  (p = ≤ 0,001)
       • Fruits secs                         OR = 0.21 (0.16–0.27)  (p = ≤ 0,001)
       • Pâtes / Féculents                   OR = 0.42 (0.33–0.54)  (p = ≤ 0,001)
       • Produits riches en MG               OR = 0.44 (0.34–0.56)  (p = ≤ 0,001)
       • Poulet                              OR = 0.37 (0.26–0.53)  (p = ≤ 0,001)
       • Activité physique                   OR = 1.94 (1.52–2.48)  (p = ≤ 0,001)
       • Œufs                                OR = 0.52 (0.40–0.67)  (p = ≤ 0,001)
       • Soupe                               OR = 0.57 (0.45–0.72)  (p = ≤ 0,001)

  3.2  Analyse multivariée (OR ajustés sur âge et sexe)
       8/24 variables significatives après ajustement (p < 0,05)
       • Sucreries                           ORa = 0.11 (0.08–0.15)  (p = ≤ 0,001)
       • Fruits secs                         ORa = 0.29 (0.22–0.39)  (p = ≤ 0,001)
       • Produits riches en MG               ORa = 0.56 (0.41–0.74)  (p = ≤ 0,001)
       • Pâtes / Féculents                   ORa = 0.57 (0.43–0.75)  (p = ≤ 0,001)
       • Activité physique                   ORa = 1.77 (1.32–2.38)  (p = ≤ 0,001)
       • Boissons énergétiques               ORa = 0.59 (0.37–0.93)  (p = 0.024)
       • Poulet                              ORa = 0.65 (0.42–0.98)  (p = 0.042)
       • Œufs                                ORa = 0.74 (0.55–1.00)  (p = 0.048)

  ⚠️  Note : Les OR < 1 pour les aliments (sucreries, graisses...) s'expliquent
      par un biais de causalité inverse : les malades déjà diagnostiqués ont
      modifié leurs habitudes alimentaires après le diagnostic. L'Activité physique
      (ORa > 1) est le seul facteur comportemental avec une direction attendue
      (sédentarité → risque accru de DT2).

6. ANALYSE EN COMPOSANTES PRINCIPALES  (ACP exploratoire)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
    • 24 variables environnementales binaires analysées
    • Variance expliquée : PC1 = 21.1%  |  PC2 = 8.9%  |  Total = 29.9%
    • La projection montre une séparation partielle entre Malades et Témoins,
      confirmant que les profils alimentaires/comportementaux diffèrent entre
      les deux groupes, sans toutefois constituer une séparation nette
      (variance cumulée modeste, typique des données binaires d'enquête).
    • Note : Pour les variables binaires, une ACM (Analyse des Correspondances
      Multiples) serait statistiquement plus rigoureuse. L'ACP reste acceptable
      à titre exploratoire.
